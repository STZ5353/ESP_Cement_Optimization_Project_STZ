# 问题一核心结果汇总

## 1. 平均出口浓度
Ridge 模型对 C_out 的 MAE 约为 0.13192 mg/Nm3，MAPE 约为 0.26513%，但 R2 接近 0，说明平均出口浓度整体稳定，工况变量对其线性解释能力较弱。

## 2. 综合除尘强度 K
CatBoost 对 K 的拟合 R2 为 0.99772，说明除尘过程存在显著非线性耦合关系。

## 3. 峰值风险
峰值点共 301 个，占比 3.0013%；峰值事件共 91 个，占比 0.9074%。CatBoost 峰值点识别 AUC 为 0.91053，Recall 为 0.73913。

## 4. 振打周期影响
T2 从约 198.64 s 增加到 257.27 s 时，峰值事件率由 0.3592% 增加到 1.4507%，约增加 4.04 倍。T4 在约 430.52 s 时峰值事件率最高，为 1.5144%。
